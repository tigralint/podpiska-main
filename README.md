# ЧестнаяПодписка

Сервис списал деньги? Онлайн-школа не возвращает оплату? Кнопка отмены спрятана за семью слоями интерфейса?

**Честная Подписка** — бесплатный правовой навигатор, который за 2 минуты сгенерирует юридически грамотную претензию на возврат денег. Без регистрации, без скрытых платежей, без сбора персональных данных.

## Что умеет

### Конструктор претензий
Два режима: возврат за подписки (Яндекс Плюс, ivi, VK) и расторжение договора с онлайн-школами (Skillbox, GeekBrains, Нетология). Документ строится на ст. 32 ЗоЗПП и ст. 782 ГК РФ — компании обычно возвращают деньги добровольно, увидев знание закона.

Выбираешь тон: вежливое обращение лояльного клиента **или** жёсткая досудебная претензия с упоминанием Роспотребнадзора и судебного штрафа 50%.

### Тренажёр отписки
Интерактивный симулятор дарк-паттернов. Попробуй найти настоящую кнопку отмены среди «Остаться с нами», «Может, передумаете?» и микроскопических крестиков. Обучение через игру — после прохождения реальные интерфейсы сервисов уже не застанут врасплох.

### Народный радар
Живая карта жалоб: какие сервисы прямо сейчас списывают деньги чаще всего. Данные обновляются из пользовательских отчётов.

### База знаний
27 пошаговых инструкций по отписке от конкретных сервисов. От Яндекс Плюс до Синергии — короткий путь сквозь все дарк-паттерны.

## Архитектура

```
views/          → страницы (Dashboard, SubscriptionFlow, CourseFlow, GuidesView...)
hooks/          → бизнес-логика (useClaimForm, useClaimFlow, useFocusTrap...)
services/       → API-клиент (geminiService, fetchWithRetry)
api/            → серверные функции (generateClaim, rateLimiter, promptBuilder)
components/ui/  → переиспользуемые компоненты (ToneToggle, FeatureCard, ToastContainer...)
store/          → Zustand-стор
types/          → TypeScript-типы
utils/          → утилиты (clipboard, errors, format, cn...)
```

Ключевые решения:
- **`useClaimForm<T>`** — generic-хук с AbortController, валидацией и retry. Один хук обслуживает оба flow.
- **`promptBuilder.ts`** — вынесенный конструктор промптов, без inline-шаблонов в API-хэндлере.
- **`RateLimiter`** — извлечённый в отдельный тестируемый класс с injectable `now`.
- **Discriminated union** для `ClaimPayload` — TypeScript гарантирует, что `calculatedRefund` есть только у курсов.

## Запуск

```bash

cd podpiska
npm install
```

Создай `.env` (пример в `.env.example`):

```env

VITE_TURNSTILE_SITE_KEY=0x...
TURNSTILE_SECRET_KEY=0x...
```

```bash
npm run dev        # фронтенд на localhost:3000
npm test           # 132 теста
npm run build      # продакшн-сборка
```

Для работы API-функций на localhost используй Vercel CLI:

```bash
npx vercel dev
```

## Безопасность

- **Turnstile** — серверная верификация каждого запроса, без капчи генерация невозможна.
- **Runtime type guards** — тело запроса валидируется на сервере до передачи в ИИ.
- **Input sanitization** — пользовательский ввод очищается от `<>{}[]` и переносов строк перед вставкой в промпт.
- **Rate limiter** — 60 запросов/час на IP. Cleanup стухших записей при каждом запросе.
- **AbortController** — при новом запросе предыдущий автоматически отменяется.
- **ErrorBoundary** — React-ошибки не роняют всё приложение, а показывают fallback.

## Зачем этот проект

Честная Подписка — социальный образовательный проект. Не стартап, не бизнес.

Идея простая: корпорации тратят миллионы на дизайнеров, которые прячут кнопку отмены. Студент, который впервые привязал карту к пробному периоду, не знает ни про ст. 32 ЗоЗПП, ни про то, что «штраф 30% за расторжение» в договоре онлайн-школы — ничтожное условие. Он просто теряет деньги и думает, что так и должно быть.

Этот проект — попытка выровнять баланс. Дать инструмент, который за 2 минуты превратит «ну ладно, бывает» в грамотную претензию с ссылками на закон.

Проект создан в рамках **Всероссийского студенческого конкурса «Твой Ход»** — как практическое решение реальной проблемы цифровой грамотности среди молодёжи.
